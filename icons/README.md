## Iconen Genereren

Je kunt iconen genereren met:

1. **Online tools:**
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/
   - https://favicon.io/

2. **Handmatig maken:**
   - Maak een vierkante afbeelding (bijv. 512x512 pixels)
   - Gebruik een bijbel of preek gerelateerd symbool (ðŸ“–)
   - Gebruik de app kleuren (primary: #2563eb, background: #f8fafc)
   - Exporteer in verschillende formaten

## Tips

- Gebruik een eenvoudig, herkenbaar ontwerp
- Zorg dat het icoon goed leesbaar is in kleine formaten
- Test het icoon op verschillende achtergronden
- Houd rekening met het "safe zone" gebied voor maskable icons
